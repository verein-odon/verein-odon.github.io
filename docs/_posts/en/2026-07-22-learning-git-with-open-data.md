We've published **Learning Git With Open Data**, a new course teaching Git and GitHub, using ODON's GitHub organisation and open data for the exercises.

The course is structured as five ~90-minute sessions, covering everything from your first commit through branching, remotes, and pull requests, with a closing module on working with open data. No prior Git experience is assumed. Handouts are bilingual (English and German).

The course was put together primarily for our interns and members, but the materials are openly licensed (CC BY 4.0), so anyone is welcome to follow along, reuse, or adapt them.

This is the first version of the course, and we'd welcome any feedback — feel free to open an issue on the [repository](https://github.com/odon-at/education-learning-git) or [get in touch](/en/contact/).
